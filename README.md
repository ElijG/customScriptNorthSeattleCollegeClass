# 🗂️ Bash File Organizer

📌 **Description:**  
This is a **Bash script** that automatically organizes files in a directory (Desktop or any folder) into respective folders based on **file type**. It ensures that your directories are neatly structured every time, making file management simple and consistent. Easily able to be expanded on or personalized. 

---

## ✨ Features

- 🖼️ **Images** → `Images` folder (png, jpg, jpeg, gif)  
- 📄 **Documents** → `Documents` folder (pdf, docx, txt)  
- 🎬 **Videos** → `Videos` folder (mp4, mov)  
- 📦 **Archives** → `Archives` folder (zip, rar)  
- 💻 **Code files** → `Code` folder (py, js, html, css)  
- 🎵 **Audio files** → `Audio` folder (mp3, wav, flac, aac, ogg)  
- 📁 **Other files** → `Other` folder  
- Interactive prompt for **custom directory input**  
- Automatically skips the script itself and handles file names with spaces
